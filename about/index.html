Theme

This site was built with the static site generator Zola. I recommend checking it out if you too like to digitally doodle on a static site as much as you like to publish on it.

Zola is flexible however majority of this site's structure stems from its theme, which was:

  1. Modified from a Zola theme, which had been
  2. Ported from a Jekyll theme called lightspeed.

Both make a great baseline to build from, so regardless of site builder being used I recommend checking them out.

The current iteration of the color scheme was heavily inspired by 100r's, Todepond's, Wryl's, and Tom MacWright's approach to their websites. I have no connection to them, but appreciate many of their ideas.
